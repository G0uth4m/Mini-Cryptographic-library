# Mini-Cryptographic-library
This library contains 4 algorithms.

[1] MD5\n
[2] SHA-256
[3] RSA
[4] Morse code

Implementation of all algorithms is in C.


## Installation :

Ubuntu :

```
$chmod +x install.sh && ./install.sh
```
OR
```
$sh install.sh
```
## Execution :
```
$./cryptlib
```
